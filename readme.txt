== Description ==

A very simple plugin that adds a drop-down listbox to the bottom of multi-page posts. The purpose of this box is to allow the user to jump directly to a named page, rather than navigating using page numbers.

== Installation ==

The plugin is simple to install:


1. Download `jump-to.zip`
1. Unzip
1. Upload `jump-to` directory to your `/wp-content/plugins` directory
1. Go to the plugin management page and enable the plugin

You can find full details of installing a plugin on the [plugin installation page](http://urbangiraffe.com/articles/how-to-install-a-wordpress-plugin/).
